# Java关键字（Java 8版本）

## 定义

- 被Java语言赋予了特殊含义，用作专门用途的字符串（单词），这些关键字不能用于常量、变量、和任何标识符的名称。

## 注意事项

- true，false和，null看起来像关键字，但它们实际上是文字; 您不能在程序中将它们用作标识符。
